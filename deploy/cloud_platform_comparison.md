# 云平台对比

| 平台 | 优点 | 缺点 | 适合场景 |
|---|---|---|---|
| 阿里云轻量应用服务器 | 国内访问稳定，文档多，适合部署 Python Web 服务 | 配置项较多，部分服务价格偏高 | 课程项目、小型后端、中文用户访问 |
| 腾讯云轻量应用服务器 | 操作简单，新手容易上手，部署方便 | 高性能配置需要额外费用 | Demo 展示、Streamlit、FastAPI |
| 华为云 | 企业级服务完整，稳定性较好 | 控制台功能多，新手学习成本较高 | 企业数字员工场景、正式方案设计 |
| Render / Railway | 连接 GitHub 方便，适合快速部署 | 国内访问可能不稳定，后台定时任务受限制 | 轻量 Demo、原型展示 |
| 本地或校园服务器 | 成本低，可控性强 | 外网访问和稳定性依赖网络环境 | 本地演示、内网部署 |

本项目推荐使用轻量云服务器，因为系统需要运行 Python、读取本地文件、保存报告，并可能扩展定时任务。
