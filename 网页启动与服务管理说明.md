## 手动启动网页

如果服务器重启后网页打不开，可以通过 SSH 登录：

```bash
ssh -p 2222 fendan@47.97.114.62
```

进入项目目录并启动网页：

```bash
cd ~/apps/digital_employee_daily_report
nohup python3 -m streamlit run app.py --server.address 0.0.0.0 --server.port 8501 > ~/apps/digital_employee_streamlit.log 2>&1 < /dev/null &
```

查看网页进程：

```bash
ps -ef | grep streamlit | grep -v grep
```

查看网页端口：

```bash
ss -lntp | grep ':8501'
```

查看网页日志：

```bash
tail -80 ~/apps/digital_employee_streamlit.log
```

## 停止并重启网页

停止旧网页进程：

```bash
pkill -f 'python3 -m streamlit run app.py'
```

重新启动：

```bash
cd ~/apps/digital_employee_daily_report
nohup python3 -m streamlit run app.py --server.address 0.0.0.0 --server.port 8501 > ~/apps/digital_employee_streamlit.log 2>&1 < /dev/null &
```

## 自动日报和周报

服务器已配置 crontab 定时任务：

```text
每天 08:00：自动采集真实技术新闻并生成当天日报
每周日 08:20：自动整理汇总周报
```

查看定时任务：

```bash
crontab -l
```

手动生成今日日报：

```bash
cd ~/apps/digital_employee_daily_report
python3 scheduler.py daily --config config.real.yaml
```

手动生成周报：

```bash
cd ~/apps/digital_employee_daily_report
python3 scheduler.py weekly --config config.real.yaml
```

查看定时任务日志：

```bash
tail -80 ~/apps/digital_employee_scheduler.cron.log
tail -80 ~/apps/digital_employee_daily_report/logs/scheduler.log
```

## 报告保存位置

最新报告保存在：

```text
reports/daily_report.md
reports/ai_claw_daily_report.md
reports/robotics_claw_daily_report.md
reports/industry_claw_daily_report.md
reports/weekly_report.md
```

每天早上 8 点生成后，会同步归档到：

```text
reports/archive/YYYY-MM-DD/
```

网页的“总结汇报”页面可以通过日期选择查看历史报告。非今日报告会显示为“归档报告，只读”。

## OpenClaw 对话修改报告

队员可以在网页的“OpenClaw 对话”页面主动要求搜索、总结和修改报告。

示例：

```text
搜索何泰然并整理机器人与具身智能方向日报段落，保存到机器人日报
```

如果对话中明确包含“修改、更新、保存、写入、生成、整理、日报、周报”等意图，系统会更新对应报告，并把旧版本备份到：

```text
reports/versions/
```

今日报告修改后，也会同步更新当天归档目录中的同名报告。

## 龙宫 bot

龙宫 bot 是 root 管理的 systemd 服务。普通 `fendan` 用户无法无密码关闭。

如需关闭，需要在服务器控制台或有 sudo 权限的终端执行：

```bash
sudo systemctl stop longgong-bot.service
sudo systemctl disable longgong-bot.service
```

检查状态：

```bash
systemctl status longgong-bot.service --no-pager
```
