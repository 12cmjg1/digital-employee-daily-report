# summarize_technical_article

## Purpose

Convert long technical text into a concise summary suitable for a daily technical report.

## Input

- title
- source
- summary
- content
- category

## Output

A short Chinese summary including the main topic, core technical point, possible value, and obvious limitation.

## Procedure

1. Read the title and content.
2. Identify the main technical topic.
3. Extract the key contribution.
4. Summarize in less than 120 Chinese characters.
5. Do not invent information not present in the input.
